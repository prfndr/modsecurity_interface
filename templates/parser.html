Анализ журнала аудита ModSecurity

При нажатие на кнопку "Analysis" производится анализ журнала аудита ModSecurity, анализируются все события до текущего момента.

Он представляется в виде:

- файл .json,

- отчет в формате .xlsx,

- .png формат с графиками.

Для этих файлов создается подпапка по пути журнала аудита ModSecurity (по умолчанию /var/log/).

При нажатие на кнопку "Show" открывается последний созданный .xslx файл состоящий из двух страниц:

- таблица, в которой каждое событие разбивается на его составные части по соответствующим колонкам,

- графики для визуализации статистики.

Графики включают в себя:

- временную шкалу с количеством заблокированных и пропущенных событий,

- круговую диаграмму с процентным соотношением входящих IP адресов,

- круговую диаграмму с процентным соотношением атак набравшие определенное количество очков аномалии,

- круговую диаграмму с процентным соотношением частоты срабатывания определенного правила.
